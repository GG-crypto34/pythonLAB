## Project Description
A web application for tracking personal expenses, managing users, and generating monthly statistics.

### Installation Guide:
1. Clone the repository.
2. Install the dependencies:  
   `pip install -r requirements.txt`
3. Run the application:  
   `python app.py`

## Technical Stack
- **Framework:** Flask
- **Database:** SQLite + SQLAlchemy
- **Methodology:** Git Flow
